CONTROLS

F1 - Show controls

MENUS

WASD / Arrow Keys - Movement
Enter - Confirm
Backspace - Back
Esc - Close

RACE

WASD / Arrow Keys - Movement
Enter - Reset to CP
Backspace - Reset to Start
R - Save incomplete replay
ESC - Pause menu

EDITOR

WASD / Arrow Keys - Move cursor
Enter - Confirm / Place block
Backspace - Back / Remove block
R - Rotate block
ESC - Pull up / Pull down piece catalogue
